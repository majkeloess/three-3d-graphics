# Grafika 3D Landing Page

Landing page z przestrzennymi tekstami 3D stworzona z Next.js i Three.js.

## Funkcje

- Przestrzenne teksty 3D z głębokością i zaokrąglonymi krawędziami
- 250 animowanych tekstów "3.0" latających w przestrzeni 3D
- Główny napis "Grafika 3D" w 3D
- Nawigacja kamery (OrbitControls)

## Instalacja

```bash
npm install
npm run dev
```

Otwórz [http://localhost:3000](http://localhost:3000)

## Technologie

- Next.js 14
- React 18
- TypeScript
- Three.js
- @react-three/fiber
- @react-three/drei
